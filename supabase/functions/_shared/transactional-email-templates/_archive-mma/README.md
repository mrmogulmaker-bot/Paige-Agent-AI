# Archived MMA / Project Mogul Enterprise email templates

These credit/funding email templates were built for the legacy Mogul Maker Academy
credit product. They are **removed from the platform default registry** (per doctrine
§2 finance prohibition and §9 platform-vs-tenant separation) but **preserved here for
reuse** by future MMA / Project Mogul Enterprise *tenant* accounts, which may register
them as THEIR own tenant templates.

Do NOT import these into the platform-default `registry.ts`.
